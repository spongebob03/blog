## Memory Process

## 단기기억의 부호화
- `Conrad`
    실험 : *발음 비슷한* 다른 글자 혼동→자극 시각적으로 제시되었어도 기억 오류 청각적 혼동 가능성있다
    - 결론 : 자극이 시각적이든 청각적이든 단기기억 부호화할때 청각적 코드 사용
- `Baddeley` : 의미정보는?
    실험 1 : 유사한 소리 나는 단어들 / 소리 구별되는 단어들 → *유사한 소리* 헷갈림
    실험 2 : 의미 유사 단어 / 의미 차이 자극 → 별 차이 없음
    - 결과 : Conrad가 맞네...
- `Shulman`
    실험 : 청각적으로 유사(Bawl,Ball) / 의미상 유사(Talk,Speak) / 리스트에 있는 단어
    - 결과 : 둘다 비슷하게 나옴 → *청각적 부호화* *의미 부호화* 모두 사용됨
- `Posner & Keele`
    실험 : 두 개의 글자 같은 글자인가 판단 
    A-a : Yes로 대답 < A-A : Yes로 대답시간 더 빠름
    - 결과 : *시각 부호화도* 안쓰는건 아니다

**결론 : 주로 청각 부호화. (시각적 부보화, 의미 부호화도 과제에 따라 일부 사용)**

## 장기기억의 부호화
**주로 의미부호화. 정보에 맞게 융통성 있게**
- `Grossman & Eagle`  
    실험 : 의미가 비슷한 방해자극에 더 많은 오경보
    - 결과 : 단어의 *의미가 비슷할 때* 혼동 가능성 더 크다 → 의미 부호화
- `Bousfield`  
    실험 : 동물15/ 직업15/ 채소15/사람이름15 뒤섞어서 제시 후 자유 회상
    - 결과 : 같은 범주에 속한 단어들을 연속적으로 회상할 확률 더 높다 → 단어들을 *의미에 따라 범주로* 묶어서 저장한 후 인출한다는 증거
- `Frost`  
    실험 : 그림 자극 이용. (의미 범주, 시각 범주(방향)) 조작 후 자유 회상
    - 결과 : 의미 정보 뿐 아니라 *시각 정보도* 부호화한다
- `Rubin`  
    실험 : 청각부호(멜로디) 이용 글(가사) 암기

- 단기 기억의 인출 Retrieval in STM
    1. 병렬 vs 순차
    2. 전체적 vs 자기종료적
    - `Sternberg`
        실험 : 숫자기억 탐지 숫자(probe) 제시 → 과제 → 반응시간 측정
        1. 얼마나 많은 숫자 기억했는가 (자극 세트의 크기)
        2. probe이 기억 리스트에 있었는가
        3. 있었다면 리스트에서 몇 번째에 있었는가
        - 결과 : 전체적 순차 처리 관찰
    
    - `Corcoran`
        병렬처리도 들어갈 수 있지 않나? → 증명 어렵
    - `Tulving & Pearlstone`
        실험 : 범주화된 단어 목록 제시
        테스트1 : 자유회상-순서에 관계없이 단어 회상 (인출)
        테스트2 : 각 범주 이름을 단서로 제시
        결과 : 단서 회상 > 자유 회상
        → 회상 과제에서의 차이는 기억의 실패가 '저장'보다는 '인출' 실패임을 암시
        
- 장기 기억의 인출
    순차, 병렬처리 모델이 공전하는 것으로 받아들여지고 있음. 과제와 사용되는 자극에 따라 달라짐
    - `Bowers, Clark, Lesgold & Winzenz`
        실험 : 인출에 미치는 범주화의 영향
        - 결과 : 조건1 (무작위 리스트) < 조건2 (범주화 리스트)
    
     
    

### 인출 실패
Why?
- 간섭이론
    - `Brown-Perterson paradigm`
        실험 : 세게의 낱자로 이루어진 문자열 제시→ 3,6,8,12,15,18초 지났을 때마다 회상 검사
        방해 : 지연시간 동안 무작위 3자리 숫자 제시, 3씩 빼는 과제 실시
        - 결과 : Rehearsal 못하게 했을 때 18초 후에 거의 망각
            → 숫자 빼기 과제가 단기기억에서의 회상 방해. 단기기억의 망각이 간섭 때문이라는 증거
            
    - 종류
        - 역행 간섭 : 이전 학습 내용 ←방해- 다음에 하는 활동
        - 순행 간섭 : 앞서 제시 된 자료 -방해→ 이후의 학습
            -부호화 시점과 인출 시점간 간격이 클 수록, 사전학습이 증가할수록 *순행 간섭 증가*
            -학습하는 정보의 종류가 달라지게 될 경우 → *순행간섭해제*
            
    - 계열위치곡선
        - 초두효과- 역행간섭을 받지만 순행간섭이 없음
        - 최신효과- 순행간섭을 받지만 역행간섭이 없음
- 소멸이론
    기억은 대체되는 것이 아니라 점차 사라진다는 입장  
    실험적 검증 어려움. ← 학습을 방해하지 않으면서 Rehearsal만 못하게 하는 과제 사용해야 함
    - `Reitman`
        실험 : 5개 단어 각 2초씩 제시. 15초간 음 탐지 과제 실시(간섭 일어나지 않는 과제)→ 단어 회상
        - 결과 : 평균 약 25% 회상률 감소
    - `Berman, Jonides & Lewis`
        실험 : 4개의 표적단어 제시 → probe 단어 제시 → probe이 표적단어 중 하나와 일치하는지 판단. 반복  
        probe :   
        -(1)완전히 새로운 단어일 때 판단시간 : baseline  
        -(2)비교적 최근 시행에 나온 단어와 일치할 때 : baseline + 50~100ms (간섭 효과)  
        -(3)더 오래된 시행에 나온 단어와 일치할 때 : (2)보다 판단지연시간 짧음. Why?  
        → 지나간 시행에 대한 단어는 rehearsal할 필요가 없음→ 소멸 가능성 큼 (소멸 효과)  
        

*간섭이 대부분의 망각 현상을 설명*

## 기억의 구성적 특성
- 다양한 책략을 사용해서 학습내용이나 경험 재구성
- 사전 경험이 정보의 회상 방식, 내용에 영향을 준다 → 구성적(constructive) 특성
    `Bransford & Johnson` : 읽은 내용이 '세탁'과 관련있다고 알고 있을 때 읽은 글 잘 기억함  
    `Bartlett` : 북미 인디언 설화 읽게 함 → 자기 마음대로 왜곡하여 글 회상   
    → 핵심 정보는 잘 회상하지만 이해하지 못하거나 애매한 정보들은 망각  
    → 사전 지식과 기대가 회상에 영향을 미침

### -처리 과정은 일반 기억과 다른가?
- Yes :`Brown & Kulik`  
    매우 충격적인 사건에 의해서만 활성화되는 과정  
    사건과 관련된 부분들에 대해서는 영구적 기록을 만들어낸다  
    과거의 위협적 사건에 대한 기억을 통해 미래의 위험을 피하도록 하는 *생존 기능*  
- No : `Neisser & Harsch`  
    다른 기억과 마찬가지로 *오류와 망각 발생*

### -정확한가?

기억의 정확도에 대한 확신은 강하지만, 실제로 다른 기억보다 더 신뢰할 수 있는지는 의심

## 기억 왜곡
기억의 구성 및 재구성 → 기억 왜곡으로 변질 될 수 있음
- `Schacter`
    - 일시성 : 기억은 신속하게 사라짐
    - 방심 : 주의를 주지 않은 사건 기억 못함
    - 차단 : 기억해야 하는데 기억이 나지 않는다 (TOT)
    - 오귀인 : 읽거나 들은 내용의 출처를 기억하지 못함 (본 적 없는데 봤다고 함)
    - 피암시성 : 외부에 의해 기억 조작. 어떤 것을 보았을 것이라는 암시 받으면 본 것으로 기억
    - 편향 : 회상의 내용이 편향되는 경향
    - 지속성 : 하찮은 일이나 별로 기억하고 싶지 않은 일 심각한 것으로 기억 (단 한번의 실패)  

- 목격자 증언
    목격자들이 기억을 복습할 떄 오류의 가능성이 매우 높다  
    특정 사건에 대한 기억은 사건 이후에 습득된 정보에 의해 왜곡되기 쉽다 (오정보 효과)  
    - `Loftus & Palmer`  
        실험 : 두 차가 충돌하는 사고 장면 → 질문 단어: "박살났을 때" > "충돌했을 때"   
        → 단순히 한 단어를 바꾸는 것만으로도 인출이 영향을 받을 수 있음
    - `Loftus, Miller, Burns`  
        목격 사건과 왜곡 정보간 간격이 증가할수록 왜곡이 더 쉽다  
        최초의 기억 흔적이 시간경과와 함께 약해지고 다른 출처에서 온 정보들로 빈틈이 채워지기 때문  
    - `Read, Connolly & Turtle`  
        암시 효과로 인해 완전히 잘못된 기억이 만들어질수 있다  
        가상의 장면이나 사건에 대해 시각적 심상을 떠올리라는 지시문 → 가짜 기억 생성 가능  
    - `Ceci & Bruck`  
        아동이 어릴수록 증언의 신뢰도는 더욱 낮아짐  
        타인의 암시를 자기의 기억으로 믿는 경우가 많다
        
    - `Kovera & Borgida`  
        목격자가 스트레스를 받는 상황에서 범죄를 목격할 때 오류 발생 가능성 높아진다  
    - `Roebers & Schneider`  
        사회적 압력이 있는 경우 목격자가 오류를 범할 가능성 높아진다

## 기억상실증 Amnesia
외현기억의 심각한 손상  
- 역행성 기억상실증  
    심한 충격을 받기 ~~전에~~ 있었던 사건들의 기억(외현기억)을 잃어버리는 증상
- 순행성 기억상실증  
    사고 이후 ~~새로운~~ 기억을 형성하지 못하는 증상
- 외현기억 : 매우 저조한 수행
- 암묵기억 : 정상 수준 수행  
    절차적 기억 과제에서는 학습에 의한 향상 보임
